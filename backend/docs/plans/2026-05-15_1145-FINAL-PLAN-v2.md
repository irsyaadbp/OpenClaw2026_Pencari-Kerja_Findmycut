## 🤖 Model Choices (Rationale)

### Vision: lucataco/moondream2
- **Kenapa:** Lightweight (2B params), fast, free tier di Replicate
- **Kemampuan:** Image captioning, VQA, object detection
- **Cocok buat:** Analisis foto wajah → extract face shape, hair properties
- **API:** Replicate HTTP API (satu platform sama image gen)

### Image Gen: black-forest-labs/flux-dev
- **Kenapa:** Open-weight, $0.03/image (paling murah dari list temen)
- **Alternatif dari list teman:** flux-2-pro ($0.05), imagen-4, flux-kontext-pro
- **Cocok buat:** Generate referensi hairstyle images berdasarkan deskripsi

### Kenapa Satu Platform (Replicate) buat Vision + Image Gen:
- Satu API key, satu base URL
- Kurang complexity di .env
- Konsisten rate limiting & error handling

## 🤖 Agent Pipeline Detail

### pipeline.ts — Orchestration

```typescript
export async function runPipeline(jobId, photoUrls, callbacks) {
  // Agent 1: Vision
  callbacks.onProgress("vision", "start", "Memulai analisis foto...")
  const features = await runVisionAgent(photoUrls, callbacks)
  callbacks.onProgress("vision", "complete", `Face shape: ${features.face_shape}`)

  // Agent 2: Knowledge
  callbacks.onProgress("knowledge", "start", "Mencari gaya rambut cocok...")
  const candidates = await runKnowledgeAgent(features, callbacks)
  callbacks.onProgress("knowledge", "complete", `${candidates.length} kandidat ditemukan`)

  // Agent 3: Ranker
  callbacks.onProgress("ranking", "start", "Menghitung skor & ranking...")
  const recommendations = await runRankerAgent(features, candidates, callbacks)
  callbacks.onProgress("ranking", "complete", "Top 3 rekomendasi siap!")

  // Agent 4: Image Gen (optional)
  callbacks.onProgress("imagegen", "start", "Generate gambar referensi...")
  await runImageGenAgent(recommendations, callbacks)
  callbacks.onProgress("imagegen", "complete", "Gambar referensi siap!")

  return { features, recommendations }
}
```

### runner.ts — Core Agent Engine

```typescript
// Autonomous loop: reason → call tool → observe → repeat
export async function runAgent(config, userMessage, onStep) {
  const messages = [
    { role: "system", content: config.systemPrompt },
    { role: "user", content: userMessage }
  ]

  let iterations = 0
  while (iterations < config.maxIterations) {
    iterations++
    onStep({ type: "thinking", message: `${config.name} berpikir... (${iterations})` })

    const response = await llm.chat(messages, config.tools)

    if (response.tool_calls) {
      for (const toolCall of response.tool_calls) {
        onStep({ type: "tool_call", message: `Memanggil ${toolCall.name}` })
        const result = await tool.execute(toolCall.params)
        onStep({ type: "tool_result", message: `${toolCall.name} selesai` })
        messages.push({ role: "tool", content: JSON.stringify(result) })
      }
      continue
    }

    return parseOutput(response.content)
  }
}
```

### Tool Definitions

#### Agent 1: Vision Tools
```
analyze_image(image_url, angle)
  → Call Replicate moondream2
  → Return: { face_shape, hair_thickness, hair_texture, ... }

merge_analyses(analyses[])
  → Majority vote + average confidence
  → Return: merged FaceFeatures
```

#### Agent 2: Knowledge Tools
```
query_face_shape_guide(face_shape)
  → Read hardcoded JSON + pgvector search
  → Return: recommended styles[]

query_hair_compatibility(style, texture, thickness)
  → Read hardcoded JSON
  → Return: { compatible, reason }

get_style_details(style_name)
  → Read JSON + pgvector
  → Return: { description, tips, maintenance, image_url }
```

#### Agent 3: Ranker Tools
```
calculate_match_score(features, style)
  → Scoring algorithm
  → Return: { overall, face_shape, hair_type, thickness, reasons[] }

generate_explanation(features, style, scores)
  → Call MiMo LLM
  → Return: { main_reason, detail_reasons[], styling_tips }
```

#### Agent 4: Image Gen Tools
```
generate_hairstyle_image(style_name, description)
  → Call Replicate FLUX-dev
  → Upload result to R2
  → Return: { image_url }
```
